# 🚀 Customer Analytics & Churn Prediction

## 📌 Project Overview
This project analyzes customer behavior and predicts churn using machine learning techniques.

## 🔧 Features
- RFM Segmentation
- Advanced Feature Engineering (Engagement Score, Avg Purchase Gap, Activity Trend)
- Churn Prediction using ML models
- Model Comparison (Logistic Regression, Random Forest, XGBoost)
- Streamlit Web App for real-time prediction

## 📊 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn, XGBoost
- Streamlit
